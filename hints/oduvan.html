Como uso o "for" para iterar sobre os índices de uma lista em Python?

Para iterar sobre cada índice de uma lista, use for i in range(len(lista)):

Isso cria um loop que passa por cada índice da lista, permitindo que você acesse e modifique cada elemento usando lista[i],

Nesse caso, i é incrementado sem a necessidade de usarmos i=i+1.

Como faço para trocar os valores de dois elementos em uma lista de maneira simples?

Você pode trocar valores entre dois elementos diretamente: lista[a], lista[b] = lista[b], lista[a]. Isso elimina a necessidade de uma variável temporária e torna o código mais limpo.

Por que, ao adicionar o estado atual da lista a uma lista de passos em um loop, todos os elementos dessa lista de passos parecem iguais ao estado final da lista original?

Isso ocorre porque você está adicionando referências à mesma lista, que é atualizada a cada passo. Para capturar e salvar cada estado separadamente, use lista[:] para criar e adicionar uma cópia da lista no momento em que o comando é executado.

Não se esqueça de adicionar somente quando realizar uma troca de elementos na lista.
